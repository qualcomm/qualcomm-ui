# @qualcomm-ui/react-mdx

React components and utilities for MDX documentation sites based on QUI Docs.

This package provides pre-built React components and utilities for creating documentation sites with MDX, including code highlighting, component demos, search functionality, and responsive layouts.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
