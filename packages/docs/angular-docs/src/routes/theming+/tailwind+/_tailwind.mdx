> [!tip]
> This guide assumes you've already [installed Tailwind CSS](https://tailwindcss.com/docs/installation/using-vite) for your application.

We provide a [Tailwind CSS](https://tailwindcss.com/) plugin which enhances your tailwind theme with QUI colors, fonts, and more.

## Install

<NpmInstallTabs packages={["@qualcomm-ui/tailwind-plugin"]} />

Once installed, import the plugin wherever you've configured your tailwind theme:

```css
@import "tailwindcss";
/* This import must be after the tailwindcss import */
@import "@qualcomm-ui/tailwind-plugin/qui-strict.css";
```
