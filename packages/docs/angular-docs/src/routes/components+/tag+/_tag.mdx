## Examples

### Icons

Icons are supplied using the [startIcon](./#startIcon) and [endIcon](./#endIcon) properties.

<QdsDemo name="TagIconsDemo" />

### Variants

Choose from three [variants](./#variant): `link`, `selectable` and `dismissable`.

<QdsDemo name="TagVariantsDemo" />

### Colors

Control the visual emphasis with different colors via the [emphasis](./#emphasis) property.

<QdsDemo name="TagEmphasisDemo" />

### Sizes

Choose from three [sizes](./#size): `sm`, `md` and `lg`. The default size is `md`.

<QdsDemo name="TagSizesDemo" />

### States

Disable the tag by setting the [disabled](./#disabled) property.

<QdsDemo name="TagStatesDemo" />

### Radius

Control the border radius of the tag with the [radius](./#radius-1) property. The default radius is `square`.

<QdsDemo name="TagRadiusDemo" />
