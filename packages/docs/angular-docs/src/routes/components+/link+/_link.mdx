## Examples

### Emphasis (color)

Links support different emphasis (colors) to match your design needs. Customize these with the [emphasis](./#emphasis) property.

<QdsDemo name="LinkColorsDemo" />

### Icons

Links can include icons to provide additional visual context. QUI links support [lucide-angular](https://lucide.dev) icons.

<QdsDemo name="LinkIconsDemo" />

### Sizes

Adjust the font and icon size of the link using the [size](./#size) property. The default size is `sm`.

<QdsDemo name="LinkSizesDemo" />

### Disabled

When [disabled](./#disabled) is true, the link becomes non-interactive and is rendered with reduced opacity to indicate its unavailable state.

<QdsDemo name="LinkDisabledDemo" />
