# {frontmatter.title}

A divider is a thin line that groups content in lists and layouts. The divider implements the [ARIA Separator role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Reference/Roles/separator_role).

import {DividerDirective} from "@qualcomm-ui/angular/divider"

### Orientation

By default, a divider has a `horizontal` orientation. You can change this by setting the [orientation](./#orientation) prop to `vertical`.

<QdsDemo name="DividerOrientationDemo" />

### Focusable

By default, a divider is not focusable. You can change this by setting a `tabindex` value. Authors can assign `aria-label` to provide accessible names to dividers.

A focusable divider [must expose a value to accessibility tools](https://www.w3.org/TR/wai-aria-1.2/#separator). Authors must set the "aria-valuenow" attribute.
