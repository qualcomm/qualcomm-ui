### Install

<NpmInstallTabs packages={["@qualcomm-ui/tailwind-plugin"]} />

Once installed, add the plugin to your Tailwind config file:

```js
/** @type {import("tailwindcss").Config} */
module.exports = {
  content: [
    // ...
  ],
  plugins: [require("@qualcomm-ui/tailwind-plugin")],
  prefix: "",
}
```
