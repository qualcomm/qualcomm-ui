# {frontmatter.title}

Due to limitations with the [font](https://developer.mozilla.org/en-US/docs/Web/CSS/font#syntax) shorthand, typeface styles are constructed from multiple CSS variables.

Each typeface uses variables for the following properties:

- `font`
- `font-stretch`

> [!tip]
> The font classes below were added in `@qualcomm-ui/styles v2.7.0`. Please update if
> you're on an older version.
