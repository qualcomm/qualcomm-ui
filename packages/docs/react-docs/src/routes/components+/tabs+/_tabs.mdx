## Examples

### Horizontal

The default [orientation](./#orientation) is horizontal. The left and right arrow keys can be used to navigate between tabs.

<Demo component={demos.TabsHorizontalDemo} />

### Vertical

In vertical [orientation](./#orientation), the up and down arrow keys are used instead.

<Demo component={demos.TabsVerticalDemo} />

### Icons and variants

Tabs support start and end icons. Customize the variant with the [iconVariant](./#iconVariant) prop. Use the [variant](./#variant) prop to change the appearance of the tab.

<Demo
  wrapperProps={{className: "bg-neutral-02"}}
  component={demos.TabsIconsDemo}
/>

### Controlled Value

The tab's active value can be controlled via the [value](./#value), [onValueChange](./#onValueChange), and [defaultValue](./#defaultValue) properties. These props follow our [controlled state](/patterns/controlled-state) pattern.

<Demo component={demos.TabsControlledValueDemo} />

### Sizes

#### Line

The line [variant](./#variant) supports four [sizes](./#size): `sm`, `md`, `lg`, and `xl`

<Demo component={demos.TabsLineSizesDemo} />

#### Contained

The contained [variant](./#variant) supports only two sizes: `sm` and `md`

<Demo
  wrapperProps={{className: "bg-neutral-02"}}
  component={demos.TabsContainedSizesDemo}
/>

### Lazy Mounted

Use the [lazyMount](./#lazyMount) and [unmountOnExit](./#unmountOnExit) props to control render behavior for inactive tabs.

<Demo component={demos.TabsLazyMountedDemo} />

### Disabled

Disable specific tabs with the [disabled](./#disabled-1) prop on the `<Tab.Root>` component.

<Demo
  wrapperProps={{className: "bg-neutral-02"}}
  component={demos.TabsDisabledDemo}
/>

### URL Search Parameters

The following example demonstrates how to render tabs that sync with the URL search parameters using react-router.

<Demo component={demos.TabsLinksDemo} />

### Context

Use the `<Tabs.Context>` component to access the tabs API from JSX.

<Demo component={demos.TabsContextDemo} />
