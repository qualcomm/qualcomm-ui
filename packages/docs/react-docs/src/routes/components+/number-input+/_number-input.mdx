### Min and Max values

Pass the [min](./#min) and [max](./#max) props to the root component to set the minimum and maximum values of the number input.

If the value entered is less than min or greater than max, it will be clamped to the nearest boundary on blur or enter.

<Demo name="NumberInputMinMaxDemo" component={demos.NumberInputMinMaxDemo} />

### Step interval

Pass the [step](./#step) prop to the root component to set the increment or decrement step interval.

<Demo name="NumberInputStepDemo" component={demos.NumberInputStepDemo} />

### States

The following shows how the TextInput component appears in each interactive state.

<Demo name="NumberInputStatesDemo" component={demos.NumberInputStatesDemo} />

### Controlled Value

Use the [value](./#value), [onValueChange](./#onValueChange) and [defaultValue](./#defaultValue) props to control the value of the input field. These props follow our [controlled state](/patterns/controlled-state) pattern.

<Demo
  name="NumberInputControlledDemo"
  component={demos.NumberInputControlledDemo}
/>

### Sizes

Customize size using the [size](./#size) prop. The default size is `md`.

<Demo name="NumberInputSizesDemo" component={demos.NumberInputSizesDemo} />

### Error Text and Indicator

Error messages are displayed using two props:

- [invalid](./#invalid)
- [errorText](./#errorText) (or the `NumberInput.ErrorText` component when using the composite API)

The error text and indicator will only render when [invalid](./#invalid) is true.

<Demo
  name="NumberInputErrorTextDemo"
  component={demos.NumberInputErrorTextDemo}
/>
