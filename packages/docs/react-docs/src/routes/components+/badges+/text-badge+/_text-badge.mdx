# {frontmatter.title}

Display text labels in compact, color-coded containers. Use text badges to categorize content, show status, or highlight information with concise labels.

```typescript
import {Badge} from "@qualcomm-ui/react/badge"
```

## Examples

### Emphasis

Use the [emphasis](./#emphasis-1) prop to convey semantic meaning or categorize content. Available semantic emphasis are: `neutral`, `brand`, `info`, `success`, `warning`, and `danger`. Color emphasis are: `blue`, `cyan`, `green`, `kiwi`, `magenta`, `orange`, `purple`, `red`, `teal`, and `yellow`.

<Demo name="TextBadgeEmphasisDemo" component={demos.TextBadgeEmphasisDemo} />

### Variant

The [variant](./#variant-1) prop controls the aspect of the badge. Two variants are available: `default` (bolder appearance) and `subtle` (muted appearance).

<Demo name="TextBadgeVariantDemo" component={demos.TextBadgeVariantDemo} />

### Size

Control dimensions using the [size](./#size-1) prop. Three sizes are available: `sm`, `md` (default), and `lg`.

<Demo name="TextBadgeSizeDemo" component={demos.TextBadgeSizeDemo} />

### Disabled

Set the [disabled](./#disabled-1) prop to visually indicate that the badge is inactive.

<Demo name="TextBadgeDisabledDemo" component={demos.TextBadgeDisabledDemo} />
