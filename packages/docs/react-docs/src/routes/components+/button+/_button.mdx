## Examples

### Variants

Buttons come in three variants: `fill`, `outline`, and `ghost`. Customize these with the [variant](./#variant) property.

<Demo component={demos.ButtonShowcaseDemo} />

### Emphasis

Control the visual [emphasis](./#emphasis) with different colors: `neutral`, `primary`, and `danger`.

<Demo component={demos.ButtonEmphasisDemo} />

### Sizes and Icons

Control dimensions with the [size](./#size) prop. Icons are supplied using the [startIcon](./#startIcon) and [endIcon](./#endIcon) props.

<Demo component={demos.ButtonSizesDemo} />

### Compact Density

Use the [density](./#density) prop to reduce the padding and height of the button. The options are `default` and `compact`.
