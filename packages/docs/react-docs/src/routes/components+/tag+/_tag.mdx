## Examples

### Icons

Icons are supplied using the [startIcon](./#startIcon) and [endIcon](./#endIcon) props.

<Demo component={demos.TagIconsDemo} />

### Variants

Choose from three [variants](./#variant): `link`, `selectable` and `dismissable`.

<Demo component={demos.TagVariantsDemo} />

### Colors

Control the visual emphasis with different colors via the [emphasis](./#emphasis) prop.

<Demo component={demos.TagEmphasisDemo} />

### Sizes

Choose from three [sizes](./#size): `sm`, `md` and `lg`. The default size is `md`.

<Demo component={demos.TagSizesDemo} />

### States

Disable the tag by setting the [disabled](./#disabled) prop.

<Demo component={demos.TagStatesDemo} />

### Radius

Control the border radius of the tag with the [radius](./#radius-1) prop. The default radius is `square`.

<Demo component={demos.TagRadiusDemo} />
