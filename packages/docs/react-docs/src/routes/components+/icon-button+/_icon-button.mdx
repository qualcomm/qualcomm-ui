## Examples

### Variants

Buttons come in three variants: `fill`, `outline`, and `ghost`. Customize these with the [variant](./#variant) property.

<QdsDemo name="IconButtonVariantsDemo" />

### Emphasis

Control the visual emphasis with different colors: `neutral`, `primary`, and `danger`. Customize these with the [emphasis](./#emphasis) prop.

<QdsDemo name="IconButtonEmphasisDemo" />

### Sizes and Icons

Customize size using the [size](./#size) property. QUI buttons come with built-in support for [lucide-react](https://lucide.dev) icons.

<QdsDemo name="IconButtonSizesDemo" />

### Compact Density

Use the [density](./#density) prop to reduce the padding and height of the button. The options are `default` and `compact`.
