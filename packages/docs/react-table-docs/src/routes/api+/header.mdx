# {frontmatter.title}

Header objects are instances representing column headers in the table. They are generated by the table and contain references to their associated column, header group, and rendering context. See the [Headers](/guides/headers) guide for usage examples.

## API

Properties and methods for header objects. Includes column references, header group information, hierarchy details (depth, index), placeholder state, and methods to retrieve rendering context and nested leaf headers.
