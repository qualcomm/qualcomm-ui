# @qualcomm-ui/typedoc-common

Common types and utilities for QUI TypeDoc integration.

This package provides shared types and utilities for generating and working with TypeScript documentation in the QUI ecosystem.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
- [Angular documentation](https://angular-next.qui.qualcomm.com/)
