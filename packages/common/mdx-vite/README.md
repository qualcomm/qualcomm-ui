# @qualcomm-ui/mdx-vite

Vite documentation plugin for applications that use MDX.

This package provides Vite plugins for building documentation sites with MDX, including support for React and Angular demos, syntax highlighting, and component documentation generation.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
- [Angular documentation](https://angular-next.qui.qualcomm.com/)
