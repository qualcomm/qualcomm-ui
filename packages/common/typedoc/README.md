# @qualcomm-ui/typedoc

Documentation generation utilities for TypeScript projects.

This package provides CLI tools and utilities for generating TypeScript API documentation using TypeDoc, with integration for the QUI documentation ecosystem.

For documentation and usage guides:

- [React documentation](https://react-next.qui.qualcomm.com/)
- [Angular documentation](https://angular-next.qui.qualcomm.com/)
